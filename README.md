# Design Humanities Timeline

BC 6세기 그리스 자연철학에서 20세기 모던디자인까지, 디자인과 인문학이 교차하는 2,500년의 인터랙티브 연대표.

**[Live Demo](https://jsonpassion.github.io/design-humanities-timeline/)**

## Overview

```
BC 6C        BC 1C       14C         18C        19C        19C말      20C초
  |            |           |           |          |          |          |
고대 그리스 → 로마/중세 → 르네상스 → 낭만주의 → 산업혁명 → 니체/프로이트 → 모던디자인
아르케        Humanitas    원근법       블레이크    수정궁      무의식       바우하우스
이데아        오캄의면도날  인쇄술       감성회복    예술공예    총체예술      3대운동
점선면        고딕/길드    지동설       3대혁명    동서양융합   페르소나      5대사조
```

## Features

- 시대별 섹션 스크롤 + 스티키 내비게이션
- 인물 카드 (호버 시 이미지 풀 노출)
- 이벤트 카드 클릭/토글 언폴드 (상세설명 + 디자인 연결)
- 호버 사이드 패널 (교차 인사이트, 중복 없는 부가 맥락)
- 시대별 배경 이미지 + 글래스모피즘
- 전체 펼치기/접기 토글
- 모바일 반응형

## Tech

Single HTML file, vanilla JS/CSS. No dependencies.
